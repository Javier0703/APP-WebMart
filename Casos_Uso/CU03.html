HOJA DE REQUISITOS

Id CU3
Nombre Buscar a los suarios
Actores Usuarios
Descripción En la aplicación podremos buscar a los usuarios y poder mirar sus perfiles, sin necesidad de entrar mediante un producto dentro de la app o desde la URL
Precondiciones
  • El usuario tiene que haber iniciado sesión
Secuencia normal
  1. El usuario clicará en el link "Usuarios" del menu que veremos arriba.
  2. Se redirigirá al usuario a la pantalla de búsqueda de usuarios.
  3. El usuario podrá buscar el perfil en concreto sin neceisdad de que sea exacto.
  4. Le aparecerá la lista de todos los usuarios que se llamen parecido a la búsqueda.
Postcondiciones
  • El usuario podrá acceder al perfil del usuario clicando en el
  • Le mostrará toda la información de ese perfil (productos, estadisitcas, informacion personal...)
Secuencia alternativa
  • Que el que hayamos puesto no exsitas en la aplicación
