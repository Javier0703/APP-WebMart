HOJA DE REQUISITOS

Id CU19
Nombre Aceptar solicitud de reserva
Actores Usuario
Descripción Cuando un usuario reserva un producto, al dueño de este le lleva una solicitud para aceptar o no ese producto
Precondiciones
  • Que exista esa solicitud
  • Puede haber más de 1 solicitud para el mismo producto
  • Podemos tener muchas solicitudes de muchos productos distintos
Secuencia normal
  1. El usuario se dirige a la pestaña de reservas dentro de su perfil
  2. Accede al panel de solicitudes
  3. Se le mostrarán todas las solicitudes que tiene
  4. Acepta las solicitudes que desee.
Postcondiciones
  • El producto quedará reservado para la persona a la que se le ha aceptado la reserva
  • Todas las solicitudes que había para ese producto se eliminan
Secuencia alternativa
