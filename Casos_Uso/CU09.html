HOJA DE REQUISITOS

Id CU9
Nombre Comprar producto
Actores Usuarios
Descripción Tendremos la opción de comprar un producto que se encuentre dentro de la app
Precondiciones
  • Que ese producto no este "reservado" para un usuario y que este libre
Secuencia normal
  1. Buscamos el producto que queremos
  2. Contactamos con el dueño de ese artículo si fuese necesario para concretar un precio y método de transporte del mismo
  3. Dar al botón de comprar artículo
  4. Esperar que el dueño acepte la compra nuestra
Postcondiciones
  • El producto será guardado como una compra de ese usuario
  • Se añadirá en el apartado de compras de nuestro perfil
Secuencia alternativa
  • Que no lo podamos comprar porque el producto esté reservado.
