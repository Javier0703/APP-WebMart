HOJA DE REQUISITOS

Id CU124
Nombre Borrar productos
Actores Administradores
Descripción Los administradores pueden borrar los productos de la aplicación si les parece que no son adecuados.
Precondiciones
  • Tengamos el rol de administrador para realizar esta acción.
Secuencia normal
  1. Buscamos los productos que queramos eliminar
  2. Le seleccionamos y le damos al botón de "Eliminar"
  3. Confirmamos la acción, ya que es irreversible
Postcondiciones
  • El producto ha sido eliminado correctamente de la aplicación
Secuencia alternativa
  • No se ha borrado el producto porque no se ha confirmado después o porque no tiene el usuario permisos suficientes.
