HOJA DE REQUISITOS

Id CU1
Nombre Login
Actores Usuario
Descripción El usuario va a iniciar sesión en la aplicacion.
Precondiciones
  • El usuario debe saber la página para iniciar sesion
  • El usuario no debe de tener sesiones activas en la aplicacion
Secuencia normal
  1. El usuario debe de encontrarse en la pagina de inicio
  2. El usuario se loguea correctamente introduciendo las credenciales:
    • Usuario
    • Contraseña
  3. El usuario le da al boton de iniciar sesión
Postcondiciones
  • El usuario se loguea correctamente.
  • Se le mostrará la pantalla inicia de la aplicacion.
Secuencia alternativa
  • El usuario no acceda porque ha fallado alguna de las credenciales.
  • Haya un fallo en el servidor/basede datos y no pueda conectarse.
