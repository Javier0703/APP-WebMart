HOJA DE REQUISITOS

Id CU07
Nombre Enviar mensajes
Actores Usuarios
Descripción En la aplicación podemos enviar un mensaje a otro usuario para la compra de uno de los productos que tiene en venta
Precondiciones
  • Estar registrado o iniciado la sesión en la aplicación
  • Que haya un producto de por medio para poder negociar, si no no se podrá
Secuencia normal
  • Se pueden diferenciar 2 casos depende del usuario:
    1. Un usuario que está en busca de un producto.
      • Buscamos un producto que esté en venta en la app
      • Le damos al botón de "Mensaje"
      • Nos saltará a un chat para poder enviarle un mensaje a ese usuario
    2. Un usuario que está vendiendo el producto.
      • Recibimos un mensaje de un usuario que quiere nuestro producto
      • Enviamos el mensaje de respuesta que queramos
Postcondiciones
  • El mensaje se ha guardado correctamente y subido a la base de datos
  • Se mostrará el mensaje que hemos enviado con la hora y el contenido de ella
Secuencia alternativa
  • No se puedan enviar o recibir mensajes por problemas en el servidor
  • No se puede enviar un mensaje porque el producto por el que se está contactando
