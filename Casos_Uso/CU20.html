HOJA DE REQUISITOS

Id CU20
Nombre Eliminar una reserva
Actores Usuario
Descripción El usuario podrá eliminar las reservas de sus productos.
Precondiciones
  • Que tenga un usuario reservado para ese producto
  • Que ese producto sea suyo
Secuencia normal
  1. El usuario accede al panel de reservas de su perfil
  2. Se dirige a mis productos reservados y selecciona el que quiere eliminar esa reserva
  3. Aparecerá un botón de eliminar reserva
Postcondiciones
  • La reserva se eliminará para se producto
  • Los usuarios podrán solicitar reservar ese producto
Secuencia alternativa
  • No se ha podido eliminar la reserva porque no era nuestra
