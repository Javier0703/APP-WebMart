HOJA DE REQUISITOS

Id CU16
Nombre Modificar una opinión
Actores Usuarios
Descripción Un usuario podrá modificar una opinion las veces que quiera.
Precondiciones
  • Debe de existir una opinion previa
  • Esa opinión debe de ser verdadera (el usuario ha comprado ese producto)
Secuencia normal
  1. El usuario se dirige al apartado de mis opiniones
  2. El usuario selecciona la opinion que desea modificar
  3. Se redirige al usuario a una página con esa opinión para cambiarla
  4. El usuario la modifica y la guarda mediante el botón de guardar que hay
Postcondiciones
  • La opinión ha sido modificada correctamente y se mostrará a partir de ahora de esa manera
Secuencia alternativa
  • No hemos podido modificar la opinión por algún fallo de conexión o porque no se ha dado al botón de guardar
