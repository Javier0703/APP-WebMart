HOJA DE REQUISITOS

Id CU08
Nombre Recibir mensajes
Actores Usuarios
Descripción Los usuarios recibirán mensajes que otros usuarios a través de un producto por el que se esté dialogando su compra
Precondiciones
  • El usuario que recibe un mensaje debe de ser mediante un char con otro usuario, con un producto de por medio
  • Se pueden recibir más de 1 mensaje del mismo usuario
  • El mismo usuario nos puede enviar mensajes desde distintos chats
Secuencia normal
  1. El usuario entra en su perfil, en el apartado de mensajes
  2. Selecciona el chat al que quiere entrar, donde ha recibido un mensaje de un usuario
Postcondiciones
  • El usuario entrará a ese chart y podrá leer el mensaje que le ha enviado otro usuario
Secuencia alternativa
  • El usuario puede no recibir ningún mensaje si no tiene ningún chat abierto, o no tiene productos por el que esos usuarios inicien un ciclo de mensajería.
